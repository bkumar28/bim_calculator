### Use Python Version
	- Python 3.8

### Create virtual environment

```
$ virtualenv venv -p python3
```

### Activate virtual environment

```
$ source venv/bin/activate
```

### install dependency

```
$ pip3 install -r requirements.txt
```

### Run application

```
$ python3 app.py
```

### Run test case
```
$ pytest test.py
```
